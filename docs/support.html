Cute Reader

Please contact me at bavan@tuberslab.com for help with the app


  • Please Allow Microphone Input
  • This app is to help a parent teach his/her kid to read using phonetics.
  • Tapping on the main board will colorize phonemes.
  • Tapping on the settigs button and completing the task will open Settings Page.
  • When you or the kid reads the word, the app listens to it and mark it.
